# Feature-Ideen für den React-Kurs

## Niedrig-Aufwand, hoher Nutzen

- **Fortschritts-Tracking** — abgehakte Lektionen im `localStorage` speichern, Häkchen in der Sidebar anzeigen
- **Suchfunktion** — Suchfeld im Header das durch Titel, Erklärungen und Code sucht
- **Schriftgröße / Dark Mode** — Toggle im Header, Präferenz im `localStorage` speichern
